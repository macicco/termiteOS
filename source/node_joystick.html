Command line:¶

miteJoy¶

Launch a joystick node

miteJoy [OPTIONS]

Options

--name <name>¶

module name

--port <port>¶

Port listen

--parent_host <parent_host>¶

Parent host to connect to. If False the node become a ROOTHUB

--parent_port <parent_port>¶

Parent port to connect to

joystick commands¶

nodes

list the children nodes:

param arg:(None)

None

param arg:(None)
